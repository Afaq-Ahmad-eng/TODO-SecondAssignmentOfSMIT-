# SMIT Backend Assignment 2 - **Full Stack Todo app**

## Project Summary

This project is a full-stack Todo application that allows users to create, read, update, and delete tasks. It demonstrates the integration of a React frontend with a Node.js/Express backend and a Prisma-managed database, following modern full-stack development practices.

## Technologies used

 - **React :** Frotnend Development
 - **ExpressJS :** Backend API's development 
 - **NodeJS :**  Runtime environment
 - **Dotenv :** For managing environment variables and securing sensitive data
 - **Prisma :** ORM (Object Relational Mapper) for database management

## Features

 - Add, edit, and delete tasks

 - View all tasks in a list

 - Backend API with RESTful routes

 - Frontend connects seamlessly with backend

 - Environment variables for security (dotenv)

 - Prisma ORM for database operations
